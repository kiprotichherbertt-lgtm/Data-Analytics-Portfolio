# Hospital Admissions Dashboard (Excel)
This dashboard analyzes hospital admissions, treatment costs, and outcomes.

## Key Insights
- Cost banding (Normal vs Expensive > 50,000)
- Department-level trends and performance
- Interactive charts for monthly admissions and cost analysis

## Files
- `Hospital_Dashboard_Excel.xlsx` – main dashboard file
- `Hospital_Dashboard_Screenshot.png` – dashboard preview

## Tools
- Microsoft Excel (Pivot Tables, Charts, Conditional Formatting)
